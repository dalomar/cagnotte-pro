# Cagnotte Pro - Gestion des Données

## 💾 Sauvegarde des Données

Vos données sont maintenant automatiquement sauvegardées dans votre navigateur, mais pour éviter toute perte, voici les solutions disponibles :

### Fonctionnalités de Sauvegarde

1. **Sauvegarde automatique** : Les données sont stockées localement dans votre navigateur
2. **Export des données** : Téléchargez un fichier JSON avec toutes vos cagnottes
3. **Import des données** : Rechargez vos données depuis un fichier exporté
4. **Suppression sécurisée** : Confirmez avant de supprimer toutes les données

### Comment utiliser

1. **Exporter** : Cliquez sur "Exporter les données" pour télécharger un fichier de sauvegarde
2. **Importer** : Cliquez sur "Importer des données" et sélectionnez votre fichier JSON
3. **Transférer** : Utilisez les fichiers exportés pour partager vos données entre navigateurs ou appareils

### Sécurité des données

- Les données restent sur votre appareil
- L'export crée une copie sécurisée
- Confirmation requise pour toute suppression
- Sauvegarde automatique en cas d'erreur

### Dépannage

Si vos données disparaissent :
1. Vérifiez que vous utilisez le même navigateur
2. Importez depuis un fichier de sauvegarde
3. En cas de problème, contactez le support
